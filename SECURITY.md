# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ALPHANSO, please report it privately rather than opening a public issue.

### Contact

Email: nelson254@llnl.gov

### What to Include

- Description of the vulnerability
- Steps to reproduce (if applicable)
- Any relevant logs or screenshots

### Response Timeline

- **Acknowledgment**: Within 5 business days of your report
- **Resolution**: We will work to address confirmed issues promptly and will keep you informed of progress

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |

## Scope

ALPHANSO is a scientific computing package. Security concerns most likely to be relevant include:
- Unsafe handling of user-provided file paths or YAML configurations
- Dependency vulnerabilities

Thank you for helping keep ALPHANSO safe.
